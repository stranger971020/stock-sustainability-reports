🎯 一、板块持续性分析 (Sector Persistence Analysis)

1. 金融原理与业务逻辑

核心假设:
持续性 = 动量效应 + 均值回归 + 趋势强度
📈 动量效应 (Momentum)
过去表现好的板块，短期继续表现好的概率更高
🔄 均值回归 (Mean Reversion)
极端高估值后会回调，极端低估值后会反弹
📊 趋势强度 (Trend Strength)
板块内部个股的协同上涨程度

2. 算法实现逻辑

核心指标计算:

指标 计算公式 金融含义 权重
收益率 (Returns) (Pt / P0 - 1) × 100 板块整体收益表现 20%
RSI (相对强弱) 100 - (100 / (1 + RS)) 超买/超卖状态 25%
MACD 信号 EMA(12) - EMA(26) 趋势方向判断 15%
趋势偏离度 (Pt - MA20) / MA20 × 100 趋势强度 15%
样本覆盖度 nstocks / ntotal × 10 数据可靠性 15%
波动率 Std(20d) / Mean(20d) × 100 风险程度 15%

3. 当前局限性 & 优化方向

问题 原因分析 优化建议
样本股票固定 硬编码 4 只代表性股票 ✅ 动态选取：按市值/流动性选取前 N 只
权重固定 人工设定固定权重 ✅ 机器学习：用历史数据训练最优权重
时间窗口单一 仅用 20-30 天 ✅ 多周期：5 天/10 天/20 天/60 天复合
板块划分简单 硬编码 12 个板块 ✅ 动态板块：用行业分类数据/聚类算法
无风险调整 未考虑 Beta/夏普比率 ✅ 加入风险调整后收益指标

📈 二、持续性个股分析 (Individual Stock Persistence Analysis)

1. 金融原理与业务逻辑

核心假设:
个股持续性 = 个体动量 + 行业协同 + 资金关注度
📊 个体动量
个股自身的价格趋势
🏢 行业协同
个股是否跟随板块同步上涨
💰 资金关注度
成交量、换手率等资金面指标

2. 算法实现逻辑

score = (
    abs(returns) * 0.30 +           # 收益率权重最高 (30%)
    abs(trend_deviation) * 0.25 +   # 趋势强度 (25%)
    (10 - min(volatility, 10)) * 0.20 +  # 波动率越低越好 (20%)
    (5 + abs(current_rsi - 50) / 5) * 0.25  # RSI 在 50 附近最佳 (25%)
)

3. 当前局限性 & 优化方向

问题 原因分析 优化建议
行业分类简单 硬编码行业归属 ✅ 使用申万/中信行业分类数据
未考虑流动性 忽略成交量/换手率 ✅ 加入资金面指标 (量比/换手率)
未考虑基本面 仅用技术面 ✅ 加入 PE/PB/ROE 等基本面指标
未考虑市值 小盘股波动大 ✅ 市值加权或分组分析

🧠 三、市场情绪分析 (Market Sentiment Analysis)

1. 金融原理与业务逻辑

核心假设:
市场情绪 = RSI 平均值 + 收益率分布 + 趋势强度
📊 RSI 均值
反映整体市场的超买/超卖状态
📈 平均收益率
反映整体市场的涨跌趋势
🎯 情绪分类
基于 RSI 和收益率的阈值判断

2. 算法实现逻辑

if avg_rsi > 70:
    sentiment = '超买'
    signal = '建议减仓'
elif avg_rsi < 30:
    sentiment = '超卖'
    signal = '建议建仓'
elif avg_return > 5:
    sentiment = '强势上涨'
    signal = '持有加仓'
else:
    sentiment = '震荡整理'
    signal = '观察为主'

3. 当前局限性 & 优化方向

问题 原因分析 优化建议
RSI 阈值固定 固定 30/70 阈值 ✅ 动态阈值：根据市场波动率调整
单一指标 仅用 RSI 和收益率 ✅ 多因子：加入 VIX、成交量、资金流向
无市场宽度 忽略上涨/下跌家数比 ✅ 加入腾落线 (ADL) 指标
